Introduction

HTML is the programming language used to code websites

HTML combined with CSS powers most of the internet

Let's learn about it

What you should already know

HTML is supported by all major browsers

HTML stands for Hyper-Text Markup Language

HTML elements

Here's a couple example elements:

  • header - creates a title for following elements
  • li - creates a list element
  • ul - creates a container for a bulleted list
  • main - creates a container for your code
  • section - creates a container for sections of a site
Hello world

The most common practice exercise for any programming language is typing 'Hello World'

You can do this by creating a p element, typing Hello World, then closing the p element

HTML Links

You can create links in your text by using anchor (a) elements

By using an anchor element you can create hyperlinks
